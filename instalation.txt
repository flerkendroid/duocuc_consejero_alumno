Instrucciones de instalacion :

1.- Copiar y descomprimir el archivo en la carpeta c:\xampp\htdocs, al final tendras una carpeta llamada “duocuc_consejero_alumno”, a la cual accederas desde el navegador como -> http://127.0.0.1/duocuc_consejero_alumno
2.- Buscar el archivo schema.sql en duocuc_consejero_alumno/schema.sql, una vez localizado copiar y pegar el contenido en una ventana de MySQL, PHPMyAdmin o MySQL WorkBench
3.- En caso de ser necesario modificar el archivo C:\xampp\htdocs\duocuc_consejero_alumno\core\controller\Database.php para los datos asociados al acceso a la base de datos.
4.- Acceder a http://127.0.0.1/furyum/admin/ y entrar al sistema con los siguientes datos.
Usuario: admin
Password: admin
